## ⚙️ التثبيت والإعداد

### 1. تثبيت المكتبات

```bash
npm install
```

### 2. إعداد ملف `.env`

أنشئ ملف `.env` في جذر المشروع وأضف المتغيرات التالية:

```env
DISCORD_TOKEN=توكن_البوت_من_ديسكورد
CLIENT_ID=معرف_التطبيق
GUILD_ID=معرف_السيرفر
DATABASE_URL=postgresql://المستخدم:كلمة_المرور@localhost:5432/اسم_قاعدة_البيانات
PREFIX=-
PORT=3000
```

> **ملاحظة:** تُنشأ جداول قاعدة البيانات تلقائياً عند أول تشغيل للبوت.

### 3. تسجيل أوامر الـ Slash

```bash
npm run deploy
```

يقوم هذا الأمر بتسجيل جميع أوامر الـ Slash (`/أمر`) مع ديسكورد للسيرفر المحدد في `GUILD_ID`.

## 🗄️ قاعدة البيانات

يستخدم البوت **PostgreSQL** وينشئ الجداول تلقائياً عند أول تشغيل.

**أبرز الجداول:**

| الجدول | الغرض |
|---|---|
| `identities` | بيانات شخصيات اللاعبين (الرصيد، IBAN، الأغراض) |
| `bank_accounts` | حسابات البنك |
| `transactions` | سجل التحويلات المالية |
| `inventory` | أغراض وحقيبة اللاعب |
| `companies` | الشركات وبياناتها |
| `company_members` | موظفو الشركات |
| `cases` | القضايا القضائية |
| `violations` | المخالفات والبانات |
| `x_posts` / `x_accounts` | منصة X (تغريدات وحسابات) |
| `snap_accounts` / `snap_snaps` | نظام سناب شات |
| `server_config` | إعدادات البوت في السيرفر |
| `admin_ranks` | رتب الإدارة ونقاطهم |
| `open_tickets` | التذاكر المفتوحة |

> يتم إنشاء جميع الجداول تلقائياً — لا حاجة لتشغيل أي SQL يدوياً.
